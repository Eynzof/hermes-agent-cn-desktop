## 关键决策摘要（详见 `01-prd` § 4）

- **项目 = workspace UI 化**，「工作室 / 档案」= profile UI 化，不增后端
- **导航**：顶导 6 项独立入口，侧栏冗余砍掉
- **v2 不做**：日报、Webhooks / 触发器、深链接、多视图变体
- **桌面原生 v2 新增**：拖拽附件、本地通知、原生菜单栏
- **首次启动**：轻引导（Preflight 清单），不强制 wizard
- **验收**：本人能用 + 挑不出毛病，不绑时间点
